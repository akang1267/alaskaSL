# ALAsKA Sign Language

teaching machines to sign

## Description

We will train a machine learning model that can translate English into 3D animated sign language. This application will be mobile and available on the App Store. The application aims to improve accessibility for the Deaf and hard-of-hearing community by providing clear and accurate sign language animations.
